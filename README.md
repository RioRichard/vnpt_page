## Technology:

- NodeJs 20.18
- MySQL (Dự kiến)
- EJS

## How to use

1. Cài đặt thư viện
``` bash
npm install
```
2. Chạy ứng dụng

``` bash
npm start
```

3. Vào browser với đường dẫn: http://localhost:3000
